Weather Dashboard

Weather Dashboard allows the user to search weather anywhere across the world. The user has access to their previous searches on the left hand side, a card containing the current search (includes an icon that describes the current weather condition) and a 5 day forecast at the footer.

Work Day Planner

The work day planner allows the user to quickly store tasks that they need to complete throughout their work day. The different colors represent current, future and past tasks, so they can manage their time appropriately.

Food for Comfort

Food for Comfort is an application that allows the user to search for food near them while viewing the weather at the same time. It is designed to save the user time. Almost always when you are going out to eat you want to know the weather as well in order to dress accordingly and seek out traveling conditions.
